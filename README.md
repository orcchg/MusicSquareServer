# MusicSquareServer

Small HTTP server able to send sample json data

API:

/**
 *  GET /all?limit=20&offset=25&genres=rap,rnb,rock
 */

/**
 *  GET /genres
 */

/**
 *  GET /genre?name=rock
 */

/**
 *  GET /single?id=100500
 */

/**
 *  GET /total?genres=rap,rnb,rock
 */

/**
 *  GET /total_genres
 */
